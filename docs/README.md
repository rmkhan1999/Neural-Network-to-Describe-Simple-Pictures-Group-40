# Neural Network to Describe Simple Pictures. Group - 40

## Folder Structure

- `src/` — source code
  - `src/data/` — data generation and preprocessing scripts
  - `src/models/` — model definitions
  - `src/training/` — training and evaluation code

- `data/` — generated data
  - `data/processed/` — final datasets used by the model
  - `data/previews/` — small example outputs for manual checking

- `docs/` — project documents
- `configs/` — config files for generation and training
- `tests/` — test scripts and checks
